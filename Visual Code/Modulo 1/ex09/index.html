    <ol>
        <li>acordar</li>
        <li>ligar para joao</li>
        <li>tomar cafe</li>
        <li>escovar os dentes</li>
        <li>ir para faculdade</li>
        <li>almoçar</li>
        <li>ir para o trabalho</li>
        <li>voltar para casa</li>
        <li>jantar</li>
        <li>dormir</li>
    </ol>

Listas não ordenadas

  • pão

    <ul>

Misturando Listas

  1. antigas
    • html
    • css
    • java script
  2. novas
    • nao tem nada
    • nao tem nada
    • nao tem nada

A lista acima ficou assim:

        
     <ol>
         <li>antigas</li>
         <ul>
             <li>html</li>
             <li>css</li>
             <li>java script</li> 
         </ul>
         <li>novas</li>
         <ul>
             <li>nao tem nada</li>
             <li>nao tem nada</li>
             <li>nao tem nada</li>
 
         </ul>
     </ol>

Usando a tag <ol> e listando os itens, usamos a tag <ul> dentro da tag <li> para fazermos uma listagem dentro do item
