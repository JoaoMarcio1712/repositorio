Código-fonte / Pré-Formatação

Para representar-mos um codigo, seja ele de html, php, pyton.. entre outros.
Podemos usar a tag <pre> (que deixa a leitura do codigo mais facil "Arrumado")
e a tag <code> (que muda a fonte das letras para que fique mais facil de entender)
siga exemplo a seguir usando os 2 codigos

        
    <!DOCTYPE html>
    <html lang="pt-br">
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge"> 
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Minhas Anotações</title>
        <link rel="shortcut icon" href="ifi.ico" type="image/x-icon" >
    </head>

Caso o contrario, o codigo ficaria assim:
<!DOCTYPE html> <html lang="pt-br"> <meta charset="UTF-8"> <meta http-equiv="X-UA-Compatible" content="IE=edge"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Minhas Anotações</title> <link rel="shortcut icon" href="ifi.ico" type="image/x-icon" > </head>
