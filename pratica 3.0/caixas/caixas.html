Utilização semantica da tag <div>

A tag <div>, é normalmente usada para criar divisões, essas divisões, por separar o conteudo, deixa mais facil a customização em css, veja aguns exemplos a seguir:

            
                <!DOCTYPE html>
                <html lang="pt-br">
                <head>
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Document</title>
                </head>
                <body>
                    <div class="titulo">

                    </div>
                </body>
                </html>

Na <div> marcada acima, se nota um parametro dentro da tag class, esse parâmetro é usado para definir um aclasse para essa tag paara que possamos edittar em css por classes, e nao por tag's,

Temos também o id, id é um identificador, geralmente usado para definir uma unica coisa, como por exemplo: "cabeçalho, corpo, rodapé"

Temos tambem as div's semanticas como "<header>" "<main>" "<footer>", que respectivamente são: cabeçalho, corpo e rodapé, assim inutilizando colocar classes ou id's com esses nomes, veja um exemplo a seguir:

            
                <!DOCTYPE html>
                <html lang="pt-br">
                <head>
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Document</title>
                </head>
                <body>
                    <main>
                    <div class="titulo">

                    </div>
                    </main>
                </body>
                </html>
