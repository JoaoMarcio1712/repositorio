Como criar botões

Passo a passo de como criar um botão

Para criarmos um botão em html5 e css3, usamos a tag <div> com uma classe expecifica para botões, e editamos sua classe para criarmos um botão
como por exemplo

Botão de teste

Para criarmos esse efeito, o codigo fica assim:

EM HTML:

            
                <div class="link">Botão de teste</div>

EM CSS:

            
                div.link{
                    display: inline-block;
                    background-color: rgb(140, 140, 218);
                    color: #000000;
                    border: 1px solid black;
                    border-radius: 5px;
                    height: 19px;
                    width: auto;
                    text-align: center;
                    
                }
                div.link:hover{
                    background-color: rgb(83, 83, 128);
                    color: wheat;
                }

Para fins explicativos eu coloquei oque cada coisa faz, é so passar o mouse em cima do que tiver duvida do que faz
