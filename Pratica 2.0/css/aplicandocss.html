Dentro do css externo, colocamos para fazer esta pagina, os seguintes parametros:

            
                main{background-color: white;
                    width: 1200px;  /*Tamanho da caixa*/
                    margin: auto;  /*Centraliza a caixa de texto*/
                    padding: 5px;  /*Margem entre a caixa de texto e o conteudo*/
                    border-radius: 5px;  /*Arredonda as pontas da caixa*/
                    }

                    body{
                        background-image: linear-gradient(180deg, black, red);  /*Prestar atenção no parametro BACKGROUND-IMAGE (não pode ser background-color)*/
                        width: auto;
                        height: 48cm;
                        background-attachment: fixed;  /*fundo imovel*/
                        font: normal bold 1em monospace;   /*font-style -> font-weight -> font-size -> font-family (SHORTHAND)*/    
                    }
