A cima eu usei uma fonte que teoricamente nao esta instalada localmente, para usar essa fonte de modo esterno, basta ir no google font, escolher uma fonta de seu gosto, e copiar o codigo de incorporação, uma linha de regra geral e outra para aplicar a fonte aonde desejar, o codigo em css ficou assim:

            
                <style>
                    @import url('https://fonts.googleapis.com/css2?family=Nanum+Pen+Script&display=swap');
            
                    h2, p1{font-family: 'Nanum Pen Script', cursive;}
                    body{font-family: Cambria, Cochin, Georgia, Times, 'Times New Roman', serif}
                </style>
