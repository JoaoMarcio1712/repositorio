Tipos de classes

Css com classes

Em css, temos dois parametros nas tag's, o ID e o CLASS, usados para fazer formatações em css em determinadas tags, no ID e na CLASS, podemos inventar qualquer valor como pro exemplo

            
                
Em HTML:

                <h1 class="titulo">
                
No arquivo Css

                .titulo{
                    e os parametros desejados
                }

Desse modo, so as tag's com o tal parametro ganhariam a formatação, ao invés de ser uma regra geral
