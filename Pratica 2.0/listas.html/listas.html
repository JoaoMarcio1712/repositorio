Trabalhando com listas

  1. Listas ordenadas

    2. Para criar-mos uma lista, usamos o codigo <ol> seguido pela tag <li>,
    Siga o exemplo das tags a seguir:

                    
                <ol type="1" start="1">
                    <li>acordar</li>
                    <li>ligar para joao</li>
                    <li>tomar cafe</li>
                    <li>escovar os dentes</li>
                    <li>ir para faculdade</li>
                    <li>almoçar</li>
                    <li>ir para o trabalho</li>
                    <li>voltar para casa</li>
                    <li>jantar</li>
                    <li>dormir</li>
                </ol>

    E a seguir a tag na pratica:

    1. acordar
    2. ligar para joao
    3. tomar cafe
    4. escovar os dentes
    5. ir para faculdade
    6. almoçar
    7. ir para o trabalho
    8. voltar para casa
    9. jantar
    10. dormir

  2. Listas não ordenadas

    3. Para fazermos uma lista não ordenada, usamos a tag <ul> seguido também pela tag <li>,
    Siga o exemplo da tag a seguir:

                    
                <ul type="square">
                    <li>pão</li>
                    <li>manteiga</li>
                    <li>Leite</li>
                    <li>alface</li>
                    <li>arroz</li>
                </ul>

    E a seguir a tag na pratica:

    • pão
    • manteiga
    • Leite
    • alface
    • arroz

  3. também conseguimos misturar as 2 listas

    4. Misturandop as 2 usando a tag <ol>, com a tag <ul> dentro dela, ou vive-versa,
    Siga o exemplo a seguir:

                        
                 <ol>
                     <li>antigas</li>
                     <ul>
                         <li>html</li>
                         <li>css</li>
                         <li>java script</li>
                     </ul>
                     <li>novas</li>
                     <ul>
                         <li>nao tem nada</li>
                         <li>nao tem nada</li>
                         <li>nao tem nada</li>
            
                     </ul>
                 </ol>

    Ficando assim na pratica:

    Minhas linguagens de programação favoritas

    1. antigas
      • html
      • css
      • java script
    2. novas
      • nao tem nada
      • nao tem nada
      • nao tem nada

  4. Temos também as listas com definições

    5. para listas de definições, usamos a tag <dl> com a tag <dt>,
    e, <dd>    , ficando assim o codigo:

                    
                    <dl>
                        <dt>HTMLe<dt>
                        <dd>Linguagem de marcação paa criação do conteudo de um sitee<dd>
                        <dt>css<dt>
                        <dd>Linguagem de marcação para a criação de design de um sitee<dd>
                        <dt>JavaScript<dt>
                        <dd>Linguagem de programação para criação de interatividade de um site</dd>
                    <dl>

    E na pratica assim:

    HTML
    Linguagem de marcação paa criação do conteudo de um site
    css
    Linguagem de marcação para a criação de design de um site
    JavaScript
    Linguagem de programação para criação de interatividade de um site
