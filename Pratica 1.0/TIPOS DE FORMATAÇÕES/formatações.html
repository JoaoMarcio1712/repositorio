Principais formatações

Nessa frase temos um termo em negrito usando a tag <b>. (não semântica)
Nesta frase, tenho um termo em destaque usando a tag <strong>.(semântica)

Italico / Ênfase

Nesta frase temos um termo em italico usando a tag <i>. (não semântico)
Nesta frase, temos um termo em ênfase usando a rag <em>(semântico).

Marcando textos

Aqui eu vou marcar uma frase como marca texto usando a tag <mark> TEXTO MARCADO
e, para mudarmos a cor do marcador, usamos um parametro de css <mark style="background-color: COR DESEJADA; "
Como nesse exemplo, usando o comando a cima, colocando a cor azul como marcador .
Caso nao coloque nenhum parâmetro de cor, a cor será automaticamente amarelo

Editando textos

Para aumentar o tamando de uma frase ou palavra usamos a tag <big> como nesse exemplo Porém.. essa tag esta obsoleta
Agora... para diminuir-mos uma frase ou palavra, usamos a tag <small> como nesse exemplo Essa tag não esta obsoleta

Texto deletado

Podemos marcar um texto como excluido para indicar q ele deve ser lido, mas nao considerado, para isso usamos a tag <del> como nesse exemplo

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, para isso usamos a tag <ins> como nesse exemplo.
Mas para só sublinhar, temos a tag <u>, porém ela ja esta na lista de tag's obsoletas

Texto Sobrescrito

Para inserir coisas do tipo x20+3. Para isso, usamos a tag <sup>, isso indica que o item selecionado deve estar a cima dos demais

Texto subscrito

Para inserir coisas do tipo H2O. isso indica que o item sekecionado deve estar a baixo dos demais

Outras formatações

Código-fonte / Pré-Formatação

Para representar-mos um codigo, seja ele de html, php, pyton.. entre outros.
Podemos usar a tag <pre> (que deixa a leitura do codigo mais facil "Arrumado")
e a tag <code> (que muda a fonte das letras para que fique mais facil de entender)
siga exemplo a seguir usando os 2 codigos

        
        <!DOCTYPE html>
        <html lang="pt-br">
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Minhas Anotações</title>
        <link rel="shortcut icon" href="ifi.ico" type="image/x-icon" >
        </head>

Caso o contrario, o codigo ficaria assim:

<!DOCTYPE html> <html lang="pt-br"> <meta charset="UTF-8"> <meta http-equiv="X-UA-Compatible" content="IE=edge"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Minhas Anotações</title> <link rel="shortcut icon" href="ifi.ico" type="image/x-icon" > </head>
