Inserindo imagens

Para usar uma imagem como icone da pagina,
usamos o comando <link rel="shortcut icon" href="NOME DA IMAGEM DESEJADA" type="image/x-icon">

Para inserir uma imagem (como eu fiz com a logo do html), eu usei a tag <img>
<img src="IMAGEM DESEJADA" alt="DESCRIÇÃO DA IMAGEM">

Imagens dinâmicas

Aqui vamos aprender a colocar uma imagem dinamica,
que se adapta ao tamanho da tela do seu usuario;
Para isso precisamos ter varias fotos de tamanhos diferentes,
de preferencia 3:

  • uma pequena(para celulares)
  • uma meida(para telas de pc que nao estão maximizadas)
  • uma grande(para telas de pc que estão maximizadas)

Aqui vou colocar um exemplo.

Para fins de teste, tente mudar o tamanho da tela do navegador para ver na pratica como funciona a imagem dinamica,
para esse tipo de imagem, usamos a tag <picture>, veja como fica o codigo:

                
        <picture>
            <source media="(max-width: 725px)" srcset="../imagens/foto-p.png" type="image/png">
            <source media="(max-width: 1025px)" srcset="../imagens/foto-m.png" type="image/png">
            <img src="../Imagens/foto-g.png" alt="imagen flexivel">
        </picture>
